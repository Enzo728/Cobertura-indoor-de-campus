# Cobertura-indoor-de-campus

Este repositorio contiene la implementación de la Práctica 2 sobre cobertura indoor en un edificio real (campus u hospital) utilizando Python y Jupyter Notebook.

## Descripción de la Práctica

**Objetivo:** Adaptar el ejercicio de penetración indoor a un edificio real donde interesa que funcionen voz, mensajería y acceso a plataformas digitales.

**Escenario:** Se estudia un edificio de varias plantas con fachada, tabiquería y un semisótano técnico.

**Herramientas:** Python, Jupyter y QGIS (para visualizaciones opcionales).

## Archivos

- `Practica2_Cobertura_Indoor.ipynb`: Notebook principal con los cálculos, tablas y visualizaciones.

## Cómo ejecutar

1. Asegúrate de tener Python instalado.
2. Instala las dependencias: `pip install numpy pandas matplotlib`
3. Abre el notebook en Jupyter y ejecuta las celdas.

## Resultados

- Tabla de pérdidas por tramo.
- Evaluación de cobertura en tres puntos: entrada, aula segunda planta, semisótano.
- Recomendaciones de refuerzo indoor si es necesario.
- Comparación de frecuencias (700/800 MHz vs 1800 MHz vs 2600 MHz).
